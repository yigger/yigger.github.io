数据结构历险记

Realloc函数介绍

先判断当前的指针是否有足够的连续空间，如果有，扩大mem_address指向的地址，并且将mem_address返回，如果空间不够，先按照newsize指定的大小分配空间，将原有数据从头到尾拷贝到新分配的内存区域，而后释放原来mem_address所指内存区域（注意：原来指针是自动释放，不需要使用free），同时返回新分配的内存区域的首地址。即重新分配存储器块的地址。

一、线性表

线性表可以说是最简单的数据结构，它的描述为：n个数据元素的有限序列。记为：L=(a1,a2,…,an)，它的样子大概长得跟下图差不多，0-4是下标，1-5是值（其实就是数组嘛）

线性表的顺序存储结构是最简单最常用的数据结构：用一段连续地址依次存储表中的数据元素。

线性表插入相当于就是在数组的某个位置插入一个值，如：
a[0] = 1; a[1] = 2; a[2] = 3; a[3] = 5;

如果要在a[1]的位置插入一个等于1.5的数，那么是不是就变成
a[0] = 1; a[1] = 1.5 a[2] = 2; a[3] = 3;a[4] = 3;

观察一下变化，很明显就是在原来的基础上，数据依次在插入的位置往后顺序移动，同理删除也是一样的，删除的元素后，依次往左移动数据

从上面可以看出，无论是插入还是删除，元素都需要依次移动（ps:删除最后一个元素不需要移动），所以时间复杂度都是O(n)
